汉学 (Sinology) 是西方学者对中国语言、文学、历史和文化的研究学科

什么是汉学？

汉学 (Sinology) 是指在中国以外地区，特别是西方国家，对中国语言、文学、历史、思想、艺术等方面进行的学术研究。

核心特征 (Core Features)

  • • 跨文化视角 (Cross-cultural Perspective)
  • • 多学科研究 (Interdisciplinary Research)
  • • 语言能力要求 (Language Proficiency)
